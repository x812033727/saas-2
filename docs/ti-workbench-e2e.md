# Ti Cloud 工作台支援手機版設定頁 E2E

## 範圍

驗證 Ti Cloud 工作台支援手機版設定頁：在手機寬度下「工作台設定」頁可閱讀、可操作，且偏好設定只保存在本機瀏覽器，不送到 Ti Worker。

## 手機版畫面重點

### 任務預設值

- 顯示說明「建立新 Conversation 時自動套用，仍可在送出前個別調整」。
- 可設定「最大成本」，畫面以美元輸入欄呈現，例如 `$ 12.5`。
- 可設定「最長執行時間」，以選單呈現，例如 `20 分鐘`。

### 外觀與版面

- 顯示主題切換，包含「跟隨系統」、「淺色」、「深色」三個選項。
- 目前選取狀態需有明確外框或高亮；手機版不應擠壓或換行錯亂。
- 顯示「預設顯示 Agent 活動」開關，說明為桌面版 Conversation 開啟右側進度面板。

### 鍵盤快捷鍵

- 顯示「快速前往全域搜尋」：`Ctrl/⌘ K`。
- 顯示「切換 Agent 活動」：`Ctrl/⌘ A`。
- 顯示「送出需求」：`Enter`。
- 顯示「需求內換行」：`Shift Enter`。

## E2E 檢查

- 以手機 viewport 開啟工作台設定頁，確認三個區塊由上到下可捲動閱讀。
- 操作成本、時間、主題、Agent 活動開關，確認互動狀態即時更新。
- 重新整理頁面後，確認本機偏好仍保留，且不需要呼叫 Worker。

## 安全原則

1. 只建立 Draft PR，不建立 Ready for review 或直接合併的 PR。
2. 每輪工作使用隔離 branch，避免把實驗或驗證變更混進主線。
3. 附件不得進入 commit；`.ticloud-attachments/` 只能作為本機參考，嚴禁複製、移動、改名後提交，或把附件內容落到文件與程式碼中。
