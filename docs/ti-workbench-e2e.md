# Ti Cloud 工作台支援手機版設定頁 E2E

## 範圍

本文件只記錄 Ti Cloud 工作台「手機版設定頁」支援狀態與安全原則；不新增響應式 CSS、不新增 Playwright E2E、不改產品行為。手機寬度下，「工作台設定」頁應可閱讀、可操作，偏好設定只保存在本機瀏覽器，不送到 Ti Worker。

## 手機版設定頁重點

1. 任務預設值：保留「建立新 Conversation 時自動套用，仍可在送出前個別調整」說明，可設定最大成本與最長執行時間。
2. 外觀與版面：保留「跟隨系統」、「淺色」、「深色」主題切換，選取狀態需有明確外框或高亮，手機版不應擠壓或換行錯亂。
3. Agent 活動：保留「預設顯示 Agent 活動」開關，說明桌面版 Conversation 會開啟右側進度面板。
4. 鍵盤快捷鍵：保留全域搜尋、切換 Agent 活動、送出需求、需求內換行等既有快捷鍵提示。

## 最小 E2E 檢查

1. 以手機 viewport 開啟工作台設定頁，確認任務預設值、外觀與版面、鍵盤快捷鍵三個區塊可由上到下捲動閱讀。
2. 操作成本、時間、主題與 Agent 活動開關，確認互動狀態即時更新。
3. 重新整理頁面後，確認本機偏好仍保留，且不需要呼叫 Worker。

## 安全原則

1. 只建立 Draft PR；未完成驗證前不得轉為 Ready for review，也不得直接合併。
2. 每輪工作使用隔離 branch，避免把實驗或驗證變更混進主線。
3. 附件不得進入 commit；`.ticloud-attachments/` 只能作為本機參考，禁止複製、移動、改名後提交，或把附件內容落到文件與程式碼中。
